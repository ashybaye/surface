Tippie College of Business Website

tippie.uiowa.edu


Description & background

  • Primary website for the Tippie College of Business, University of Iowa
  • Collaborated with the marketing and design company to create the content strategy and design comps. Then, a team of three developers was formed to implement the Drupal 7 solution. The development of the website took 8 months; the website was launched in August 2016.

Features & technology used

Theme

  • Created a custom theme based on the Radix base theme.
  • Used extensively many features of the Bootstrap framework that come with Radix including SASS, JQuery, Breakpoints, as well as SASS mixins and Flexbox.
  • Helped to create the style guide document for the website's editors.
  • * Three options for page header - no image, short image, a tall image with custom text, a tall image with the Story reference.
  • Next Steps footer block: created a content type Calls to Action to display referenced CTAs in the footer. Some CTAs implement a lightbox (see the footer on the homepage, click on Request Info).
  • Customized CKEditor, added custom CKEditor styles and widgets.

Homepage

  • Custom Panels page with programmatically created Panel panes.
  • Used Youtube API for the lightbox video player controls.
  • Developed functionality to display the urgent message with the "curtain" effect.
  • You'll Love It Here panel: Created a custom Panel pane with the carousel of featured content cards using Slick module API
  • Used exposed filter view block for instant program search ("Explore Programs" panel)
  • Used Instagram Social Feed module to embed the stream of photos from the College's Instagram account

Structure

  • Created 19 content types
  • Implemented Paragraphs module with 17 bundles
    • Background style selection
    • Taxonomy and entity references to enable re-use of content (see Student Story and Up and coming paragraphs here)
  • Created several custom landing pages using Panels and Views (ex: Program Finder, News, Event Calendar)
  • Created custom module Tippie Alerts for displaying urgent/important information on top of every page in case of emergency.
  • Moved the MBA Events Calendar from Coldfusion to Drupal.

Integrations

  • Digital Measures (faculty, staff, and graduate students profiles for the People Directory)
  • Act-On (custom Request Info forms that post data to ActOn and Salesforce)
  • Social networks (Twitter, Facebook, Instagram)
  • * Localist (Events)
  • XML and RSS feeds (ex: list of recruiting events in the Up and coming block; Jobs page)
  • Google Structured Data
  • Custom implementation of the GDPR module (using ip geo-location, custom javascript, GTM configuration).

John Pappajohn Entrepreneurial Center Website

iowajpec.org

Description & background

John Pappajohn Entrepreneurial Center (JPEC) is the entrepreneurial hub of the University of Iowa that provides a variety of services and programs for students, businesses, and the general public to support, educate, and facilitate collaborations.

I was the project manager and the only developer for this website. I worked with the JPEC's communication team to plan content, design custom pages, and translate marketing requirements into easy-to-use Drupal 7 solution. The development took 6 months; the website was launched in August 2017.

Highlights & features

  • JPEC theme is based on the custom Radix theme with Tippie branding
  • Homepage elements are fully customizable
  • Upcoming Events block on the homepage is implemented using Slick Carousel module
  • 10 paragraph bundles were created
  • Three different options for the header
  • Events Calendar based on the Calendar module
  • Extensive use of WebForms module to accommodate events registrations
